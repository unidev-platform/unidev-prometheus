# unidev-prometheus

Library to expose Prometheus metrics

# Usage

```
...
repositories {
	maven { url 'https://jitpack.io' }
}
...
dependencies {
	implementation('com.github.unidev-platform:unidev-prometheus:0.0.1')
}

```
